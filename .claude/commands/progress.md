학습이나 갱신 없이 진도만 보여준다(읽기 전용).

1. `tutor/PROGRESS.md`를 읽는다.
2. 요약 출력: 세션 카운터, 전체 진행(✅/🟡/⬜ 개수와 %), 단계별 진행(1️⃣~7️⃣), 현재 위치.
3. 복습 예정 큐에서 밀린 노드(있으면)를 나열.
4. 다음 할 일 추천: 밀린 복습이 있으면 `/review`, 없으면 `/learn`으로 다음 노드(들) 안내.

이 커맨드는 어떤 파일도 수정하지 않는다. 한국어로 간결하게.
